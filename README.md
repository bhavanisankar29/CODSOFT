# CODSOFT Java Tasks

This workspace contains basic Java console projects for different tasks.

## Task1 - Number game
- A simple guessing game where the user tries to find a randomly generated number.

## Task2 - Student grade calculator
- Calculates total marks, average percentage, and final grade based on subject scores.

## Task3 - ATM interface
- Simulates basic ATM operations such as checking balance, depositing money, and withdrawing money.

## Task4 - Currency converter
- Converts an amount from one currency to another using live exchange rate data from an API.

## Task5 - Student management system
- Menu-driven student record management with add, edit, remove, search, display, and file-based persistence.
